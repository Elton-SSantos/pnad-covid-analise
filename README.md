# pnad-covid-analise
Análise exploratória e inferencial da PNAD COVID-19 2020 com foco em emprego e acesso à internação hospitalar (comparação público vs privado). Projeto desenvolvido em R.

# PNAD COVID-19 — Análise de Impacto no Emprego e Acesso à Internação

## Sobre o Projeto

Este projeto analisa o impacto da COVID-19 sobre:

- Situação de emprego
- Acesso à internação hospitalar
- Comparação entre hospitais públicos e privados

Utilizando a base de dados da PNAD COVID-19 2020.

O objetivo é explorar padrões socioeconômicos e desigualdades no acesso à saúde durante a pandemia.

---

## Tecnologias Utilizadas

- R
- tidyverse
- ggplot2
- rmarkdown

---

## Estrutura do Projeto

- `01_tratamento.R` → Limpeza e transformação dos dados
- `02_graficos.R` → Construção dos gráficos
- `pnad_covid_analise.Rmd` → Relatório final com análise interpretativa

---

## Principais Análises

- Taxa de desemprego durante a pandemia
- Comparação de internação público vs privado
- Análise descritiva e visual

---

## Objetivo Profissional

Projeto desenvolvido com foco em análise de dados na área da saúde, aplicando boas práticas de organização, reprodutibilidade e visualização.
